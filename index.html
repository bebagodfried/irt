How to codebook

  • Compiler:
    • gcc main.c -o bin

  • Compile & execut:
    • [linux/macOs] gcc main.c -o bin && ./bin [windows] gcc main.c -o bin && bin.exe

  • Compile, linker & execut:
    • [linux/macOs] gcc main.c -lm -o bin && ./bin [windows] gcc main.c -lm -o bin && bin.exe
